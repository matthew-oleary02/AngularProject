# AngularProject

Simple Angular web application for customer records (create, read, update, delete) using a SQL Server database.

## How to Run

1. Clone repository
2. Open the project in VS Code
3. Ensure `Node.js` installation
4. Run backend with `node server.js` command in backend directory
5. Run frontend with `ng serve` command in customer directory
5. Open your browser and go to `https://localhost:4200/customers`
6. Frontend UI allows for managing of customer records

## Notes

- backend directory contains .env file and server.js file
- ensure .env file has correct credentials to connect to local SQL Server database
